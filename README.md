# Employee Attrition Analysis

This project analyzes employee attrition using Python (Jupyter Notebook) and visualizes insights using a Power BI dashboard.

## Tools Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook
* Power BI

## Project Overview

The goal of this project is to understand factors influencing employee attrition such as age, salary, overtime, job role, and department.

## Project Workflow

1. Data Collection – Loaded the employee attrition dataset.
2. Data Cleaning – Handled missing values and prepared features.
3. Exploratory Data Analysis (EDA) – Analyzed patterns across age, salary, overtime, job role, and department.
4. Modeling – Applied parametric and non-parametric models to study attrition behavior.
5. Visualization – Built an interactive Power BI dashboard to present insights.

## Project Structure

data/ – Dataset used for analysis
notebooks/ – Jupyter Notebook analysis and modeling
dashboard/ – Power BI dashboard file
images/ – Dashboard screenshots

## Key Insights

* Attrition rate is around 16%.
* Employees working overtime show higher attrition risk.
* Lower salary groups show higher attrition.
* Early career employees are more likely to leave.
