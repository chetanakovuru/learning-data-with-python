# Employee Attrition Analysis

This project analyzes employee attrition using Python (Jupyter Notebook) and visualizes insights using a Power BI dashboard.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
- Power BI

## Project Overview
The goal of this project is to understand factors influencing employee attrition such as age, salary, overtime, job role, and department.

## Project Structure
data/ – dataset used for analysis  
notebooks/ – Jupyter Notebook analysis  
dashboard/ – Power BI dashboard file  
images/ – dashboard screenshots

## Key Insights
- Attrition rate is around 16%
- Employees working overtime show higher attrition risk
- Lower salary groups show higher attrition
- Early career employees are more likely to leave
